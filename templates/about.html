R Notifications

  • Version: 1.0.0
  • Developed: with the flask framework
  • Screen support: Desktop and Tablets via Web Browser

    • R Notifications is an application designed to create push notifications using the OneSignal SDK. It was developed as part of the final project for the CC50 course at Harvard EAD, completed in 2024 and promoted by Grupo Estudar in Brazil. The application was built with Flask/Python, with the aim of simplifying the testing process when implementing notifications, making life easier for developers.

    The source code is available in a public repository so that it can be downloaded, modified and distributed, as long as due credit is given to the author, Marcos Fabiano Correia Rangel.

    Feedback on errors, suggestions for improvements, constructive criticism and compliments are very welcome and can be sent via email in the contacts section, where channels are available for this purpose.
