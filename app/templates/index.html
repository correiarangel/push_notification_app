Principal Content

The Push Notification App is a practical tool to facilitate initial testing with OneSignal and also for production use when sending push notifications.

Benefits:

Initial Testing and Production: Allows you to easily create and trigger custom notifications, simply by entering the App ID and API Key provided by OneSignal. Cache Clearing: We recommend clearing the cache and session data when exiting, especially in production environments, to avoid security issues. Attention in Production: Once OneSignal is active for your end users, use the tool with caution to avoid testing in real environments.

⚠️ Important: Since it is hosted on a free service, the tool may be unstable or temporarily offline.
