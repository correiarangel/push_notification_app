About Sistym

Push Notification App

  • Version: 1.0.0
  • Developed: with the Flask framework
  • Screen support: Desktop and Tablets via Web Browser

Push Notification App is an application designed to create push notifications using the OneSignal SDK. It was developed as part of the final project for the CC50 course at Harvard EAD, completed in 2024 and promoted by Grupo Estudar in Brazil. The application was built with Flask/Python, with the aim of simplifying the testing process when implementing notifications, making life easier for developers.
